# Test Worktree Project

This is a test project for demonstrating Git worktree functionality.

## Overview

This repository contains basic HTML and CSS files for testing purposes.

## Structure

- `index.html` - Main HTML file
- `index.css` - Stylesheet for the HTML page

## Usage

Open `index.html` in a web browser to view the page.

## Development

This project uses Git worktrees for managing multiple branches simultaneously.
